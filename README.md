## Compilação
Para compilar, executar o comando 
```
> nmake
```
ou
```
> nmake all
```
Serão criados os executáveis `diskexp.exe` e `filegen.exe` no diretório corrente.

Este programa deve ser executado no sistema operacional Windows, versão 10 ou superior.
